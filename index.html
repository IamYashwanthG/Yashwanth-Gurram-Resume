Restful Detector

The RESTful detector integrates LoRaWAN, BACnet, and Modbus protocols to gather data from various sources. This data is transmitted via MQTT protocol.To ensure seamless interaction and integration with other services, it authenticates the user to secure access and the system utilizes Swagger API to offer RESTful services, allowing for standardized and easy-to-use endpoints.

PI Tag Creation

The Tulip application was developed to manage OSI PI Tags, which are collected from various devices and stored within the OSI PI system. This application provides functionalities to create, delete, duplicate, and update these tags efficiently. Additionally, it facilitates the pushing of updated tags to the database, ensuring that data remains current and accurately reflects device inputs.

Track and Trace

The Track and Trace Tulip application is designed to monitor items throughout an industrial process by scanning QR codes at each station. This application enables real-time tracking of an item’s progress through various stages of production or assembly. Additionally, it provides a comprehensive history of the item, documenting its journey and any interactions it undergoes. By leveraging QR code scanning and detailed historical tracking, the application enhances visibility, accountability, and efficiency in the production process.

Andon Management

The Andon Management Tulip application is used in industrial settings to monitor and manage production lines in real time. It provides a visual and interactive system for reporting and addressing issues such as equipment malfunctions or quality concerns. Through the application, operators can trigger Andon alerts, which are visual signals (like lights or screens) that indicate the need for immediate attention. This tool facilitates quick response to problems, improves communication across teams, and enhances overall production efficiency by ensuring that issues are promptly identified and resolved.
